# ❌⭕ Tic Tac Toe Game

A classic Tic Tac Toe game built using HTML, CSS, and JavaScript. This project provides a simple and interactive game experience with a clean user interface and smooth gameplay.

---

## 🚀 Features

- ❌⭕ Two-player gameplay  
- 🎯 Interactive game board  
- ⚡ Real-time move updates  
- 🔄 Game reset functionality  
- 🎨 Clean and responsive UI  

---

## 🎮 How It Works

1. Open the application  
2. Player 1 starts with ❌  
3. Player 2 plays with ⭕  
4. Players take turns marking the grid  
5. The game detects win or draw  
6. Reset to play again
